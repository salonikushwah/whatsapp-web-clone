# 💬 WhatsApp Web Clone (React.js)

## 📌 Project Overview
This project is a WhatsApp Web-like chat application built using React.js.  
It allows users to select contacts, send messages, and view chat history.  
All messages are stored in the browser using localStorage.

---

## 🚀 Features
- 📋 Contact List (Sidebar)
- 💬 Chat Window
- ✍️ Send Text Messages
- 🖼️ Send Images
- 😊 Emoji Support (UI)
- 💾 Data stored using LocalStorage
- 🔄 Messages persist after refresh

---

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6)
- Context API (Global State)
- React Hooks (useState, useReducer, useEffect)
- CSS (Flexbox)

---

## ⚙️ Setup Instructions

### Clone the repository:
```bash
git clone https://github.com/salonikushwah/whatsapp-web-clone.git
```

### Go to project folder:
```bash
cd whatsapp-web-clone
```

### Install dependencies:
```bash
npm install
```

### Run the app:
```bash
npm start
```
